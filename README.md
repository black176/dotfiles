# Dotfiles

Personal dotfiles, is created an link in $HOME for each item inside this folder

## Requirements

make emacs st tmux i3 slock i3status dmenu xsel feh fish-shell brightnessctl

## Clone
```sh
git clone ssh://git@github.com/zbioe/dotfiles
```

## Setup Links

```sh
./setup_links
```
or
```sh
make setup_links
```
